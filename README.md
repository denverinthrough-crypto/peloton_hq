\# Peloton HQ



A cycling race simulation and management web app built with Django.



\## Features

\- Simulate professional cycling races (Monuments and Grand Tours)

\- Rider profiles with skill ratings (sprinting, climbing, cobbles, endurance)

\- Realistic race results with time gaps and race day noise system

\- Supports races like Paris-Roubaix and Tour de France

\- Team and rider management



\## Tech Stack

\- Python / Django

\- PostgreSQL

\- Bootstrap

\- HTML / CSS



\## How to Run

1\. Clone the repo

2\. Create a virtual environment: `python -m venv venv`

3\. Activate it: `venv\\Scripts\\activate`

4\. Install dependencies: `pip install -r requirements.txt`

5\. Create a `.env` file with your database credentials

6\. Run migrations: `python manage.py migrate`

7\. Start server: `python manage.py runserver`
